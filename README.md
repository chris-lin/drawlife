# 繪塗人生 (Draw Life) #

「繪塗人生」是一個具有塗鴉板功能的聊天室，是「Taiwan Up Up 好好玩」團隊參加 Nodeknockout 2012 的參賽作品。

更多的介紹請參考參賽網頁： http://nodeknockout.com/teams/taiwan-up-up
